✅ Version 1: Core & Clean (MVP – Minimum Viable Product)
🎯 Goal: Let people read Qur’an and access Adhkar offline, beautifully.
🔧 Features:
📖 Qur’an reader:

Arabic + optionally translation

Scroll surahs, simple clean UI

Local SQLite for ayahs

🤲 Adhkar section:

Morning / evening / sleep / prayer

Organized by category

Can use local JSON or SQLite

🏠 Home Tab:

Welcome message (du'a, hadith)

Daily verse or dhikr

💰 Payment:
No payment yet

Only show a small note:
“This app is free. Later versions will include optional support and donations, insha’Allah.”

🚀 Version 2: Motivation Layer
🎯 Goal: Make the experience fun and spiritual, without counting hasanat.
✨ New Features:
🧩 Challenges tab:

“Read 1 page today” ✔️

“Read 3 dhikr from morning set” ✔️

Show checkmarks or completion bar

📆 Daily streak system:

Show a “7-day streak” (optional local storage)

🎁 Simple badges:

“7-day reader”, “Daily dhikr hero” (not too gamified, keep it minimal)

🔒 Payment:
Add subscription screen: $0.99/month

Add “I can’t afford it” button (unlocks everything)

Message on screen:
“This helps support the app. If you can’t afford it, you’re still welcome here — it's between you and Allah.”

🎤 Version 3: Memorization Mode (Your Unique Feature)
🎯 Goal: Introduce mic-based recitation with feedback.
🧠 New Features:
🎙️ Recite mode:

User selects surah

Mic listens to each ayah

Beep if not correct (basic matching via Google Speech-to-Text)

🧘 Repeat and retry

✅ Simple progress tracker: shows surahs they’ve practiced

Technical Note:
You can start with simple exact match of words (no need for tajweed grading at this stage). Use SpeechRecognizer in Android, compare text, and add small tolerance.

🤲 Version 4: Donation/Sponsorship + UI Polish
🎯 Goal: Let users support others easily.
❤️ New Features:
“Buy & sponsor 1 free user” option

“This free access was gifted by someone” message

Track sponsor_count & free_unlocks_used

Add themes or visual unlocks (optional)

Refine UI animations, feedback

🔮 Later Versions (Optional Future Ideas)
📱 Full tajweed color display

📊 Memorization analytics

🧑‍🤝‍🧑 Friend invite system with sponsor code

🌐 Multi-language support (French, Urdu, etc.)

📅 Custom dhikr reminders
